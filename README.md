# 🎨 AI Data Analyst Dashboard – Frontend

## 🚀 Live Demo

🌐 Frontend:https://ai-sales-dashboard-frontend.vercel.app/
🔗 Backend API: https://ai-sales-dashboard-backendend.onrender.com

---

## 📌 Overview

The AI Data Analyst Dashboard is a modern web application that allows users to upload CSV data, analyze it using AI-powered queries, and visualize insights through interactive charts.

This frontend provides a clean and intuitive interface for interacting with the backend analytics engine and AI system.

---

## 🛠 Tech Stack

* **Framework:** Next.js (React)
* **Styling:** Tailwind CSS (Glassmorphism UI)
* **Charts:** Recharts
* **State Management:** React Hooks
* **API Communication:** Fetch API

---

## ✨ Features

### 📁 File Upload

* Upload CSV files easily
* Clean UI with custom upload component
* Displays selected file name

---

### 📊 Data Visualization

* 📈 Line Chart (Trend analysis)
* 📊 Bar Chart (Comparative analysis)
* Dynamic data rendering

---

### 🤖 AI Assistant

* Chat-based interface
* Accepts natural language queries
* Example:

  * "Total sales"
  * "Top 5 products"
  * "Sales by region"

---

### 🧠 Smart Insights

* Auto-generated insights
* Highlights best/worst performers

---

### 🎨 Modern UI/UX

* Dark theme
* Glassmorphism design
* Blur background effects
* Interactive cards & hover animations

---

## 🔗 API Configuration

Create `.env.local` file:

```env
NEXT_PUBLIC_API_URL=https://your-backend.onrender.com
```

---

## ▶️ Run Locally

```bash
npm install
npm run dev
```

Open:

```
http://localhost:3000
```

---

## 🧠 How It Works

1. User uploads CSV file
2. Data is sent to backend
3. Backend stores data in MongoDB Atlas
4. User enters query (natural language)
5. AI converts query → structured JSON
6. Backend processes JSON → returns result
7. Frontend displays charts and insights

---

## 📂 Project Structure

```
client/
├── app/
│   ├── page.tsx
├── components/
├── styles/
├── public/
├── package.json
```

---

## 🌐 Deployment

* Frontend deployed on **Vercel**
* Backend deployed on **Render**
* Database hosted on **MongoDB Atlas**

---

## ⚠️ Known Limitations

* No authentication (yet)
* Limited support for complex multi-condition queries
* Depends on backend availability

---

## 🚀 Future Improvements

* Add authentication (JWT / Clerk)
* Export reports (PDF/Excel)
* Real-time analytics
* Mobile responsive UI
* Advanced filtering (multi-condition queries)
